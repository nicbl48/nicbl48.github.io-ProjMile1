HTML

Hypertext Markup Language

HTML stands for Hypertext Markup Language. HTML is basically the foundation of the internet and its webpages. While many websites use CSS and Javascript for more of the flasher applications, HTML is the basic building blocks of the webpages. As there have been different iterations of HTML, it has always been fundamental part of the internet that everyone relies on.

The HTML tag will encapsulate an aspect of the webpage in opening and closing angle brackets (<,>). An example of such tag is h. This tag is used to define the header of a page or structure of the website heading. The HTML code is made up of all the tags and is defined with an opening and closing tag.

The elements of HTML are text that are encapsulated by the tag themselves. An example is h This is the website headline /h. Other common tags are p for paragraph, img for images, audio for audio and so on. The tags can have other characters to further instruct the browser during interpretation.
