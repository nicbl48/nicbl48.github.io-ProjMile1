Tim Berners-Lee

HTML creator.

The first version of HTML was written in 1991 by Tim Berners-Lee. This version was known as HTML 1.0. He created it while working at the European group CERN. Engineers needed a better way to share information between computers. Tim Berners-Lee came up with an idea for an internet-based hypertext system. This would end up being the foundations for HTML

While at CERN in 1990, Tim Berners-Lee and a fellow colleague, data systems engineer Robert Cailliau, came up with an official request for funding from CERN to work on creating this hypertext system. CERN unfortunately denied this request from the two.

Later, the first publicly accessible concept of HTML was a document called “HTML tag”. Tim Berners-Lee would reference this document on the internet in 1991. This document was a very simple concept of the HTML that Tim Berners-Lee made or the one that we use today. It was influenced by in-house languages at CERN and still has some characters that are in the current iteration of HTML.

Tim Berners-Lee is considered to be the 'father of the internet'. His work and research were instumental to how we still develope the web
